# Frequently Asked Questions

### What is an inReach device?

[These](https://discover.garmin.com/inreach/personal/) are small handheld "satellite communicators" which allow users (with an active [subscription](https://www.garmin.com/p/837461)) the ability to send tracking and text/email messages from remote areas, unserved by other means of communication.

### What is MapShare? How do I set it up?

[MapShare](https://support.garmin.com/?faq=p2lncMOzqh71P06VifrQE7) is an inReach feature which allows friends or loved ones the ability follow progress in real time online.

### What does this plugin Do?

This plugin requests data from your MapShare page and embeds it anywhere Shortcodes are supported.
